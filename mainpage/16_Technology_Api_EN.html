HACİME GÖRE İŞLEM KOMİSYONLARI

THE API [APPLICATION PROGRAMMING INTERFACE]

API is the interface which grants the functions of an application to be accessed and used both externally or remotely. It also allows the application on a server to be accessed from different platforms and to generate transactions by responding to the queries requested. Icrypex allows you to access the information you need with relevant API queries. Thus, you can trade and monitor the price and volume of cryptocurrencies without connecting to the Icrypex platform itself.

WHITE LABEL API DOCUMENTATION

These documents contain the information required for Icrypex Prime's white label platform API. These documents are for the Icrypex API's and other Icrypex Prime partners.

USAGE

Add your stock exchange's URL to the beginning of the methods used in order to use the API.

For instance, you can check the following links to get information from Icrypex:

  • [https://www.icrypex.com/api/ticker]
  • (https://www.icrypex.com/api/prices)

TEST

In order to test the API's, you can use our test platforms. The balances displayed in the test area are not real and does not represent any actual value. Test platforms run on Bitcoin TESTNET and you can deposit your TESTNET money into your account.

Test Platform:

  • [IcrypexTest] (https://test.icrypex.com)

IMPORTANT

You will not be able to use the test platforms on our mobile application. They are only synchronized with our live platforms. Please use the Inquiries section of the github project to report your questions and issues.

Limits:

  • .../api/ticker requests are limited to 10 requests per 100 milliseconds.
  • Other requests are limited to one every 100 milliseconds.
  • If you attempt to make more than 50 unauthorized entries, yourIP address will be blocked

Visit the Github page for the Icrypex API.
