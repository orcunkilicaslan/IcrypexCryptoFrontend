BITCOIN HAKKINDA

Bitcoin is a first digital currency created in January 2009. BTC transactions are made by digitally exchanging anonymous, heavily encrypted hash codes across a peer-to-peer (P2P) network. Each user’s Bitcoins are stored in a program called a digital wallet, which also holds each address the user sends and receives Bitcoins from, as well as a private key known only to the user.
