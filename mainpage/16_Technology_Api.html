WHITE LABEL EXCHANGE API DOCUMENTATION

The documentation for Icrypex Prime's white label exchange platform API.These docs are for the APIs of Icrypex and other Icrypex Prime partners.

USAGE

Append your exchange's URL to the beginning of the methods to use the API.

For example, to get ticker info from Icrypex, use:

  • [https://www.icrypex.com/api/ticker]
  • (https://www.icrypex.com/api/prices)

TESTING

You can use our testing platforms to test the APIs. The balances on the test sites are not real and do not represent any real value. The testing platforms work on Bitcoin TESTNET and you can deposit TESTNET coins to your account.

The testing platforms are:

  • [IcrypexTest] (https://test.icrypex.com)

IMPORTANT

Our mobile applications will not work with the testing platforms. They can only be synced with our live platforms. Questions & ProblemsPlease use the issues on this github project to ask questions and report bugs.

Request Limits:

  • .../api/ticker requests are limited to 10 requests per 100 miliseconds.
  • Other requests are limited to 1 request per 100 miliseconds.
  • If you make more than 50 consequent unauthorized requests your IP address will be blocked.
